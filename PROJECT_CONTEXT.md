# 프로젝트 개요

## 목적

판매처와 면세점에서 받은 Excel/PDF 주문 자료를 분석하여 회사 표준 품목명과 품목코드로 변환하고, 택배 출고 파일 및 이카운트 창고이동 전표를 만드는 프로그램이다.

## 사용 흐름

1. 사용자가 Supabase 계정으로 로그인한다.
2. 프로그램이 품목, 세트 구성, 별칭, 바코드, 담당자 및 창고 정보를 불러온다.
3. 주문 Excel/PDF를 선택하거나 드래그 앤 드롭한다.
4. 입력 양식을 B2C 또는 B2B로 자동 판별한다.
5. 품목을 정확/유사/확인필요/미등록으로 분류한다.
6. 사용자가 필요한 행을 수정한다.
7. 택배 출고용 Excel로 변환하거나 이카운트 창고이동으로 등록한다.

## 입력 유형

- B2C: 셀메이트 주문 파일. 열과 행의 수가 변해도 헤더 이름을 검색하여 읽는다.
- B2B: 시티, 롯데, 신라, 신세계, 현대 등 면세점 출고 요청 파일.
- 창고이동: 품목명 또는 품목코드와 수량이 있는 Excel/PDF.

## 주요 소스

- `main.py`: GUI, 로그인, DB관리, 출고 및 창고이동 화면
- `excel_loader.py`: 셀메이트 Excel/CSV 분석
- `duty_free_loader.py`: 면세점 Excel 및 바코드 분석
- `matcher.py`: 출고 품목과 세트상품 매칭
- `ecount_transfer.py`: 창고이동 파일 분석, 품목 매칭, 이카운트 API
- `shipping_export.py`: 택배 출고 Excel 생성
- `REQM_1_0_7.spec`: 통합 Windows EXE 빌드 설정

## 권한

- 관리자: 품목 DB 추가·수정·사용중지, 별칭 저장
- 일반 사용자: 조회, 분석, 행 단위 수동 수정 및 출고 변환
- 실제 권한은 Supabase `app_user_roles`를 기준으로 판단한다.
